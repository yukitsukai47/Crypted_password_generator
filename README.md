# Crypted_password_generator
パスワードを自動生成し、テキストファイルに出力します。
その後、出力したテキストファイルを暗号化します。
復号にはcrypto.pyのオプション-dを使用して、行います。

# 動作環境
mac or Linux
Python 3系（python3.8確認済み）

# 使い方
python3 password_generator.py
